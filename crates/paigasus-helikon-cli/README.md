# paigasus-helikon-cli

Command-line binaries (`helikon` and `paigasus-helikon`) for the [Paigasus Helikon](https://github.com/SMK1085/paigasus-helikon) AI SDK — a Rust SDK for building AI agents.

> **Binary-only — not published to crates.io as a library.** Build the binaries from source. They are currently stubs; the real CLI lands in a future release.

## Build & run

```bash
cargo run -p paigasus-helikon-cli --bin helikon
cargo run -p paigasus-helikon-cli --bin paigasus-helikon
```
