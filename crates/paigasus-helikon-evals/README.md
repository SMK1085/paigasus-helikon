# paigasus-helikon-evals

The evaluation harness for the [Paigasus Helikon](https://github.com/SMK1085/paigasus-helikon) AI SDK — a Rust SDK for building AI agents.

> **Not yet implemented.** This crate is a pre-published `0.0.0` name-claim placeholder: the name is reserved on crates.io and the [`paigasus-helikon`](https://crates.io/crates/paigasus-helikon) facade exposes an `evals` feature for it, but the crate is currently empty. **Don't depend on it yet** — the real implementation lands in a future release.

Track progress in the [crate roster](https://smk1085.github.io/paigasus-helikon/reference/crates.html) and on [the documentation site](https://smk1085.github.io/paigasus-helikon/concepts/observability-evaluation.html).
